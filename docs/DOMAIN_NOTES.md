# AuDHD Focus Tool – Domain Notes

## What this app is for

This application is designed for people with autism/ADHD (AuDHD) to help them in keeping track of different projects and activities and the state(s) of those projects/activities. A user can toggle different projects to different focus modes, setting certain ones as active/foregrounded and others as background or inactive. Along the way, the user can also make notes about where they left off on different projects or record observations about what remains to be done.

## Core problem

For me, as a neurodivergent individual, I operate according to certain MODES: ex, reading mode, creative writing/brainstorming mode, developer mode, worker mode across gig platforms, etc. When I switch from one mode to another, I often have to go back and re-acquaint myself with where I left off, which can be cognitively expensively, time-consuming, and generally confusing, as information and state get stored across a variety of sites, such as ChatGPT conversations, bookmarks, Notion boards, etc. This application will create a visual layer that helps me more easily ground where I left off via "mode" setting and organization for projects.

## What this app is not

This isn't a kanban board/Trello-type task application. There may be states like "in-progress" or "done," similar to such an application, but the point of this application is more to emphasize modes, particularly primary modes. For example, when I'm in writing mode/working on my creative writing, the UI should actually bring this into focus on the screen and have some visual indicator (ex: darkening other modes) to indicate I'm actively working on writing.
